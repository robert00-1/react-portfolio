# Title 
My portfolio

# Description
 react portfolio platform is a single react application that allows users to showcase projects dynamicaly, and search through existing projects. The application is reponsive and bult using modern react practices.

 # Tools
Vite
CSS
React
Javascript

# Setup
follow this instruction to access and run react-portfolio
clone the repository
git clone https://github.com/robert00-1/react-portfolio.git
- navigate into the project folder
cd react-portfolio
- install dependancies
npm install
npm run dev
 copy and paste the URL that will appear after running the npm run dev into your browser to see and access the the react-portfolio

 # Additional Instructions
